Data Vatsalya - Prompt Engineering Event

Description

It is an engaging technical event where students use prompt-based engineering and vibe coding to generate desired outputs. It encourages creativity, logical thinking, and hands-on problem-solving through innovative problems, blending AI-assisted coding with analytical reasoning to enhance participants' technical skills and real-world application abilities in a fun, competitive environment.

Guidelines

Participants will be given challenges that require crafting prompts, writing small code snippets or prompts + code combinations, and iterating to achieve specified outputs. Evaluation will consider correctness, creativity, efficiency, reproducibility, and explanation of approach. Teams/individuals should document their prompt-engineering steps and provide a short demo of the output.

Outcome

Participants enhance their prompt engineering, coding and problem-solving skills. They gain hands-on experience in generating AI-driven outputs, improve logical reasoning, and develop confidence in using technology effectively. The event nurtures innovation, teamwork, and real-world application of data and coding concepts.
