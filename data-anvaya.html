Data Anvaya - No-Code ML Event

Description

Participants will build machine learning models without coding using Teachable Machine or Google AutoML. The challenge tests their ability to tune models efficiently within limited time, promoting understanding of automated ML workflows, creativity, and problem-solving through user-friendly AI tools.

Guidelines

Teams will be provided datasets or can use allowed datasets to create models on no-code platforms. Participants should focus on data preparation, feature selection (as supported), model selection/options, hyperparameter tuning (where available), and clear presentation of evaluation metrics. Demonstrations must include reproducible steps and explanation of choices.

Outcome

Participants will gain practical exposure to automated machine learning platforms, model evaluation, and optimization. They will develop analytical and teamwork skills, learn the basics of ML pipeline tuning, and understand how no-code tools simplify AI model development for real-world applications.
